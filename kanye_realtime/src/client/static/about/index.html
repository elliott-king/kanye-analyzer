Predicting comments

I hand-classified ~1000 comments, and created a simple ML model based on them. It makes guesses in real-time on the wavy feed page. I really appreciate when you classify a comment, as it will make the model better!

It's accuracy is mediocre, but it is significantly better than a random guess (10%). The production model is a simple Naive Bayes Classifier, although I have a Multilayer Perceptron that I may switch it with. You can see the code for them at my Github (below).

Categories

I took user comments containing the word 'wavy,' or containing ðŸŒŠ classified them as either positive or negative. Then, I classified them into one of the following categories:

  • poster: another poster on the subreddit
  • op: the original poster of the thread
  • link: the subject of the thread
  • this_sub: r/kanye as a whole (or all its users)
  • Kanye: the man himself
  • external_person: an external individual (other than Kanye)
  • external_object: an external object
  • self: the user posting the comment
  • copypasta
  • misc
